# PythonFromZero

A simple project to learn python in a very easy way, ment for people who can't program at all and want to learn it with python

All the Level Details and what you have to do is in the `Aufgaben` folder and are only available in German.

## 

The Solution for every Level are on this Branch. You can also find more information for the "teacher"
in the file [Teacher.md](./Teacher.md)

## Studen Code

The branch for the students with all the templates ready to solve are on the `Student` Branch.
