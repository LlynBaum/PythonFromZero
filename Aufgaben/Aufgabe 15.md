# Aufgabe 15

In dieser Aufgabe schauen wir uns Zahlen nochmals genauer an. Natürlich können wir mit Zahlen auch rechnen.
Die zeichen die verwenden werden um zu rechnen sind fast gleich wie bei der Mathe in der Schule:

- Addition => `+`
- Subtraktion => `-`
- Multiplikation => `*`
- Division => `/`

Jetzt kannst du einfach wie in der Mathematik deine Rechnung schreiben. Eine rechnung kannst du einfach wie eine Zahl irgendwo in dein program schreiben.
Als kleines Beispiel `1 + 1`.

## Aufgabe

Wieder mithilfe von `think`, lass dir das Resultat anzeigen von folgender rechnung `<dein alter> + 10`.

Mach weitere Beispiele für jede Operation die es in der Mathematik gibt, also:

- Addition => `+`
- Subtraktion => `-`
- Multiplikation => `*`
- Division => `/`
