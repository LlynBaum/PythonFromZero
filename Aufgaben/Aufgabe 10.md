Die Idee ist: "Es wäre doch cool, wen es `move_diagonal()` gäbe!". Das gibts es allerdings nicht, aber man kann das auch
einfach selber machen, mit hilf von methoden.

## Methoden

Eine Methode ist ein Code Block, den man mit dem Namen ausführen kann. Eine Methode erstellen kannst du wie folgt:

```python
def name():
    # hier kommt dein code
```

Dabei ist der name frei wählbar, mit der einzigen bedingung, es darf keine Leerschläge darin vorkommen.
Die Methode kannst du dann wie folgt verwenden:

```python
name()
```

**Hinweis:** In zukunft wen du eine eigene Methode erstellst, darfst du diese Kopieren und in neuen Aufgaben verwenden,
wen du willst. Das ist auch das Ziel von Methoden, Code wieder verwenden zu können :)
