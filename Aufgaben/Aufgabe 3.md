## For Loop

Ein loop im Programmieren ist ein Code, der mehrmals ausgeführt wird. Es gibt verschiedene Arten von Loops, wir fangen jetzt mal mit dem for loop an.
Und so sieht ein `for loop` in python aus:

```python
for i in range(n):
    # hier kommt dein code
```
Der Code unterhalb des Kommentars wird `n` mal ausgeführt. Also z.B. um etwas 2-mal auszuführen, schreibst du anstelle von `n` eine 2.
