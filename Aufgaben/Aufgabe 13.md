# Aufgabe 13

Für diese Aufgabe musst du folgendes machen:

1. Mache einen Loop, der den Character bewegt bis er am Ziel ist.
2. Im Loop werden folgender Code ausgeführt (in dieser reihen folgen)
   1. `try_move_to_side()`
   2. `move_right()`
3. Erstelle die `try_move_to_side()` methode
4. (Challenge) Achte darauf das der Character nie in eine Wand hinein Laufen will

## Hinweis

Du kannst auch ein `if` in einem `else` machen:

```python
if condition:
    # code
else:
    if condition2:
        # code2
```

Das kannst du auch kurz schreiben als:

```python
if condition:
    # code
elif:
    # code2
```
