## While Loop

Wie wir gelernt haben, der `for` loop wiederholt etwas ein bestimmte anzahl. Der `while` wiederholt etwas auch mehrmals
aber anstelle von einer gewissen anzahl, macht wiederholt er den code solange eine Bedingung zutrifft.

```python
while bedingung:
    # hier kommt dein code
```
